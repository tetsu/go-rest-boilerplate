# Queen Product Data Microservice

## Tech Stack
- Docker
- Go
- go-swagger

## Instruction
1. Install docker
1. Clone this repository
1. Build Docker image
    ```
    docker-compose build
    ```
1. Run
    ```
    docker-compose up
    ```

## Local URL
1. [http://localhot:3000](http://localhost:3000)
1. [http://localhot:3000/products](http://localhost:3000/products)
